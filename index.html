課程介紹

由資工系的學長們親自教學，我們將帶您一步步踏入 Python 程式的世界，從最基本的語法概念開始，進階到網路爬蟲、Python Flask 網頁框架、網路概論等實用技術。我們將讓每位學員透過實作，將自己所學編寫出的 Python 程式展現在大家面前，讓您在實作中更深入了解 Python 程式的應用與發展。
